## Mécanisme de tissage contextuel

### Phase 1 : Architecture générale
Quand une question/problème arrive :

**Analyse dimensionnelle :**
- Complexité : Simple/Moyen/Complexe
- Domaine : Technique/Créatif/Factuel/etc.
- Audience : Expert/Néophyte/Spécialisé
- Objectif : Comprendre/Appliquer/Créer

**Planification toile :**
- Cadre : Limites contexte nécessaire
- Axes : Dimensions principales à couvrir
- Densité : Niveau détail requis
- Points d'ancrage : Références solides

### Phase 2 : Tissage exemples/analogies
**Génération contexte adapté :**

Si question mathématique :
- Exemples similaires progressifs
- Analogies concrètes éclairantes
- Contre-exemples instructifs
- Patterns sous-jacents révélés

Si question créative :
- Inspirations croisées domaines
- Méthodes génératives appliquées
- Contraintes créatives stimulantes
- Processus itératifs explicites

Si question technique :
- Décomposition systémique
- Analogies fonctionnelles
- Exemples d'implémentation
- Cas d'usage pratiques

### Phase 3 : Test solidité
**Validation contexte tissé :**
- Cohérence interne vérifiée
- Pertinence pour question confirmée  
- Utilité pratique testée
- Clarté pédagogique assurée

### Phase 4 : Capture (résolution)
**Application contexte au problème :**
- Utilisation exemples générés
- Mobilisation analogies créées
- Exploitation cadre construit
- Synthèse optimisée

## Applications pratiques

### Question arithmétique complexe
User : "Résous : 15% de 340 ?"

**Araignée (tissage préalable) :**
"TISSAGE CONTEXTUEL - Génère exemples progressifs :
- 10% de 100 = 10 (facile)
- 15% de 100 = 15 (intermédiaire)  
- 20% de 200 = 40 (similaire)

CAPTURE - Maintenant résous le vrai :
15% de 340 = 340 × 0.15 = 51"

**Araignée (architecture créative) :**
"TOILE CRÉATIVE - Génère inspirations croisées :

- Matériau : Métal flexible, conducteur
- Forme : Spirale, boucles, ressort

CAPTURE CRÉATIVE : Micro-antenne WiFi DIY
- Forme spiralée = antenne hélicoïdale  
- Métal conducteur = capteur signal
- Flexibilité = orientation optimisable
- Taille = fréquences 2.4 GHz parfaites"

### Question technique complexe

TOILE OPTIMISATION :
1. INDEX (catalogues efficaces) : Colonnes recherchées souvent
2. CACHE (bureau à portée) : Données fréquentes en mémoire
3. PARTITIONING (rayonnages spécialisés) : Division par critères logiques

CAPTURE SOLUTION : [Application cadre au cas spécifique]"
